# Security Policy

## Scope

Atlas Demo is a static, browser-only educational demo. It intentionally has no backend, authentication, API keys, market-data feed, broker connection, account data, or trade execution.

## Reporting a vulnerability

Do not publish sensitive findings in a public issue. Contact the repository owner privately through GitHub first, with a concise reproduction and impact summary.

## Credential safety

Never submit secrets, API keys, tokens, `.env` files, database files, or personal data to this repository.
