# Portfolio Research

## Purpose

Study portfolio intelligence methods that can improve allocation, risk
awareness, diversification, and recommendation fit.

## Candidate Technologies

- Mean-variance optimization
- Risk parity models
- Factor exposure models
- Drawdown-aware allocation rules
- Black-Litterman style views
- Constraint solvers for portfolio construction

## Evaluation Criteria

- Risk-adjusted return improvement
- Turnover and transaction sensitivity
- Constraint support
- Interpretability for users
- Behavior under concentrated holdings
- Runtime and dependency footprint
- Fit with existing portfolio health scoring

## Integration Status

Research only. No live trading, broker integration, or automated rebalancing is
included.
