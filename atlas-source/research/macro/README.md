# Macro Research

## Purpose

Track macroeconomic data and models that could contextualize Atlas market
status, sector risk, rates sensitivity, and regime detection.

## Candidate Technologies

- FRED economic data
- Treasury yield curve data
- Inflation and employment datasets
- Regime classification models
- Sector rotation indicators
- Macro-aware risk scoring

## Evaluation Criteria

- Data freshness and reliability
- Historical availability
- Signal stability across regimes
- Explanatory value for recommendations
- API cost and limits
- Maintenance requirements
- Fit with market status and risk engines

## Integration Status

Research only. Atlas does not currently run macro model integrations.
