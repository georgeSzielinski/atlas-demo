# Backtesting Research

## Purpose

Define evaluation methods for measuring whether Atlas recommendations perform
well over time and under realistic constraints.

## Candidate Technologies

- Vectorized backtesting libraries
- Event-study frameworks
- Walk-forward validation
- Benchmark-relative return analysis
- Portfolio-level simulation
- Slippage and transaction-cost models

## Evaluation Criteria

- Determinism and reproducibility
- Support for recommendation timestamps
- Handling of holding periods and expirations
- Benchmark comparison support
- Runtime on local history
- Clarity of hit rate and return metrics
- Ease of connecting to validation results

## Integration Status

Research only. Atlas has a validation foundation and backtest engine, but no
full trading simulator or execution workflow.
