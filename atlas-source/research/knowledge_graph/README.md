# Knowledge Graph Research

## Purpose

Explore how Atlas could connect companies, sectors, filings, news, events,
macro factors, and portfolio exposures into a queryable research graph.

## Candidate Technologies

- NetworkX
- Neo4j
- RDF and SPARQL stores
- Entity resolution pipelines
- Embedding-enhanced graph retrieval
- Company and sector taxonomy datasets

## Evaluation Criteria

- Entity resolution quality
- Relationship explainability
- Query performance
- Storage and operational complexity
- Fit with evidence generation
- Ability to cite source records
- Maintenance burden as data sources change

## Integration Status

Research only. Atlas does not currently persist or query a knowledge graph.
